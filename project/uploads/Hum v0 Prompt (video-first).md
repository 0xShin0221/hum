# Hum — 動画主役ランディングページ 作り直し用プロンプト（v0 / コピペ可）

> **使い方:**
> 1. このファイルの内容を丸ごと v0 の入力に貼る。
> 2. 続けて **自分で作成した動画ファイルを添付**し、「**この動画をヒーローに使ってください**」と一言添える。
> 3. 別案がほしいときは末尾「Creative latitude」の軸を1つ変えるだけ。

---

## 0. 一言で
**Hum** の超イケてる、**動画が主役**のランディングページ（1枚もののインタラクティブな実装）を作って。
**添付した動画**をヒーローで**全画面（フルブリード）**に使い、その上にコピーとCTAを重ねる。
主役は「**ハンズフリーで話して使うAI**というコンセプト」と「**その体験を映した動画**」。

## 1. プロダクトの本質（絶対に外さない）
Hum は、**Bluetoothイヤホン（AirPods等）越しに、手も画面も使わずAIと“話して”使えるデスクトップアプリ**（Mac / Windows）。
話したアイデア・会話・ミーティングを**ぜんぶ覚えていて、いつでも呼び出せる**。
- ❌ イヤホン／ハードウェアを売る製品ではない。イヤホンは入口にすぎない。
- ✅ 物語の主役は「**一日中、ただ話しかけるだけで、記録も要約もリマインドもしてくれる**」体験。

## 2. このページの唯一の仕事
訪問者がファーストビュー（スクロールなし）で「**話すだけでいい。いつでも、ぜんぶ残る**」と体感し、
アプリを**ダウンロード（Mac / Windows）**したくなること。

## 3. 動画の扱い（ここが最重要 — 動画が主役）
- **添付された動画をヒーローの背景として全画面で再生**する。実装は必ず:
  `<video autoPlay loop muted playsInline preload="auto" poster={ポスター画像}>`。
  （`muted` と `playsInline` が無いとモバイルで自動再生されない。必ず付ける）
- 動画の**上に半透明のスクリム（グラデーション）**を重ね、どんな明るさの映像でもコピーが読めるようにする。
  例: 下から `theme背景色 → 透明` の縦グラデ＋左から軽い横グラデ。**文字は動画に焼かず、必ずHTMLで重ねる**。
- 動画ファイルパスは**1か所の定数/props にまとめる**（例: `HERO_VIDEO_SRC`）。差し替えが一発でできるように。
- ライト/ダークで別グレードを出し分けたい場合は `heroVideo.light` / `heroVideo.dark` の2スロットを用意（無ければ同一でOK）。
- 動画がまだ無い箇所（ユースケース等）は **`<video>` に存在しない src を直書きして404を出さない**こと。
  代わりに「**ストライプのプレースホルダ＋モノスペースの説明ラベル**」（例: `use-case video · commute`）を置く。
- **アクセシビリティ**: 自動再生は `muted` 前提。`prefers-reduced-motion: reduce` のときは動画を一時停止し、ポスター画像（静止）にフォールバック。再生/一時停止のトグルも添える。

## 4. ファーストビュー（スクロールなしで完結）
- 全画面ヒーロー動画＋スクリムの上に:
  - バッジ: 「AIボイスアシスタント」/ "AI Voice Assistant"（先頭にエメラルドの小さな点 = 「AIが聞いている」）
  - 見出し（**1行**・超特大エディトリアル）: 「AI、いつも耳元に。」/ "AI, always in your ear."
  - サブ: 「イヤホンを通じてハンズフリーで会話。アイデア・会話・ミーティング — すべてを記憶。」
  - CTA 2つ: 「Mac版をダウンロード」「Windows版」（Apple / Windows のグリフ付き）
  - 下端に控えめなスクロール誘導。

## 5. 必須セクション
1. ナビ（ロゴ＝**音波モチーフ**／機能／使い方／言語切替／テーマ切替／Download。スクロールで背景がブラー＋実体化）
2. ヒーロー（上記・**全画面動画が主役**）
3. ユースケース（「いつ使う？」を3〜4シーン。**通勤・会議後・料理中・移動中**。動画/プレースホルダを左右交互に配置＋短い説明）
4. 主要機能 4つ（2×2グリッド）:
   - **Always Listening** — 話しかけるだけ。手も画面もいらない。
   - **Perfect Memory** — すべての会話を記憶、いつでも検索。
   - **Meeting Notes** — 自動で要約、議事録を生成。
   - **Quick Capture** — ひらめきを瞬時に、後で整理。
5. クロージングCTA（「耳元に、もう一つの脳を。」＋ダウンロード。背景に脈動するエメラルドのオーブ）
6. フッター（ロゴ／リンク／言語／著作権）

## 6. 体験・インタラクション
- **i18n**: 言語切替UIで実際にコピーが切り替わる。最低 **日本語＋英語**、できれば 中/韓/西/仏/独 も。デフォルトは日本語。選択は永続化（localStorage）。
- **ダークモード**: ライト=白、ダーク=**真っ黒(#000)**。トグルで全要素対応＆永続化。動画も light/dark で出し分け可。
  （注意: テーマ変数を `transition` した状態で切り替えると一部の色が古い値で固まるブラウザ挙動があるため、**切替の瞬間だけ全 transition を無効化**してから適用する）
- 上品で繊細なモーション（Apple調イージング `cubic-bezier(0.22,1,0.36,1)`）。
  スクロール連動で**ヒーロー動画が僅かにスケール**、セクションは静かにフェードアップ。過剰な装飾はNG。
- 「AIが聞いている」サインとして、**脈動するエメラルドのオーブ／音波**をページ全体の“糸”として要所に効かせる。
- レスポンシブ（モバイルファースト）。CTAはモバイルで全幅、もしくは下部固定バーでも可。

## 7. アエステティック（強い初期値 — 大胆に再解釈OK）
- プレミアム・ミニマル（Apple製品ページ調）。白ベース＋真っ黒ダーク。たっぷりの余白、超特大見出し、繊細なシャドウ。
- 色は3〜4色に制限：白・黒・グレー＋**アクセント1色（エメラルド `#12B76A`）**。
- タイポ: **Geist**（SF Pro近似のクリーンなサンセリフ）。日本語は **Noto Sans JP**。見出しは詰めたトラッキング・行間0.95前後。
- プレースホルダは**ストライプ＋モノスペースのラベル**で受ける。**SVGでイヤホンや人物を描き込まない**。

## 8. 技術・スタック（v0向け）
- **Next.js (App Router) + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion** で実装。
- 構成はコンポーネント分割（`Nav` / `Hero` / `UseCases` / `Features` / `ClosingCTA` / `Footer` / `LanguageProvider` / `ThemeProvider`）。
- i18n はページ内の辞書オブジェクト＋Contextで再現（重い国際化ライブラリは不要）。
- ダークモードは `next-themes` でも自前でも可。`class` ベース。
- **コンソールエラーを出さない。実在しないファイルを `src` に直書きして404を出さない**。
- 画像/動画が必要な箇所は props or 定数でパスを受け、**添付動画は1か所差し替えで反映**できるように。

## 9. Creative latitude（“作り直し / 別案”。前回と差をつける）
**1つの大胆な方向にコミットして**。次のどれかを軸に振り切ると別案になる:
- レイアウト: フルブリード動画ヒーロー / 分割2カラム / 縦スクロールの映像ストーリーテリング。
- タイポ: 超特大エディトリアル / 静かで小さめ・余白で語る。
- 色: エメラルド維持 or **別の単色アクセント**を1色だけ（彩度・明度は揃える）。
- モーション: スクロールで動画がスケール／キャプションが流れる／オーブが脈動。
- 構成: 「シーン→機能」を交互に編む / 1スクリーン=1メッセージのフルスクリーン連続。
※ コンセプト（ハンズフリー音声AI・**動画主役**・ダウンロード誘導・イヤホンを売らない）と
　制約（1行見出し・余白・アクセント1色・文字を動画に焼かない）は維持。

## 10. 仕上げ
1. まず「想定・コンセプト・採用するビジュアル方向」を短く宣言してから着手。
2. **ファーストビュー（全画面動画＋コピー＋CTA）を最優先**で形にして早めに見せる。
3. **添付した動画をヒーローに配線**（`HERO_VIDEO_SRC` 一発差し替え）。ポスター画像とreduced-motionフォールバックも用意。
4. ダウンロードCTA・言語切替・ダークモード・動画の自動再生が**実際に動く**こと。

### 付録: 添付するヒーロー動画の推奨仕様
- 内容: 日常で“ハンズフリーに話して使う”瞬間（通勤しながら／会議を出ながら／料理しながら 等）。スマホは見ない・持たない画。
- 各シーンで耳元にエメラルド `#12B76A` の光が脈動するとブランドが繋がる。
- 自然光・浅い被写界深度・スロー・Appleライフスタイル調。**画面内テキストなし／音声なし**。
- 形式: MP4(H.264) もしくは WebM。シームレスループ。フルブリード前提なら横長(16:9)、縦長配置なら9:16や4:5。
- 容量はWeb最適化（数MB〜十数MB目安）。ポスター用に1フレームの静止画も書き出しておく。
